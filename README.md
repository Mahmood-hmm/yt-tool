## yt-dlp 
It is a yt-dlp based tool to make install videos much much esiers

## Requirments:
This tool needs a yt-dlp // ffmpeg // pyhon

## installation
1) Download ffmpeg and yt-dlp

`sudo apt install ffmpeg`

`sudo apt install yt-dlp`

2) Download tool

`git clone https://github.com/Mahmood-hmm/yt-tool.git`

`cd yt-tool`

`sudo chmod a+rx yt-tool.py`

`./yt-tool.py`
