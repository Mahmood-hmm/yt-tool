
## yt-dlp 
It is a yt-dlp based tool to make install videos much much esiers

## Requirments:
This tool needs a yt-dlp // ffmpeg // pyhon

## Installation for linux :
1) Download ffmpeg and yt-dlp

`sudo apt install ffmpeg`

`sudo apt install yt-dlp`

2) Download tool

`git clone https://github.com/Mahmood-hmm/yt-tool.git`

`cd yt-tool`

`sudo chmod a+rx yt-tool.py`

`./yt-tool.py`
## Installation for windows :
1) Download python
2) Download python file
3) Install yt-dlp and ffmpeg in exe and put in the same folder
4) Run cmd in same folder then `type pip install rich`
5) Type `python yt-tool.py`
